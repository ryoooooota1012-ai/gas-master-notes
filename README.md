# ガス主任技術者試験 学習資料

ガス主任技術者試験（甲種）の学習資料・過去問をまとめたリポジトリです。

## フォルダ構成

```
.
├── 基礎/          # 基礎科目の過去問（平成27年度〜令和5年度）
├── ガス技術/       # ガス技術科目の教材と過去問
│   ├── 製造/      # 製造分野の教材・分野別過去問
│   └── 過去問/    # 年度別過去問
└── 法令/          # 法令科目の教材と過去問
    └── 過去問/    # 分野別・年度別過去問
```
